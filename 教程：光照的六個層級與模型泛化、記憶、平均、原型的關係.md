# 光照的六個層級與模型泛化、記憶、平均、原型的關係

為什麼月光在水面上的質感可以很美，但鏡子裡的倒影總是歪的？

為什麼模型參數越大，反而更容易產出平庸的結果？

seed 到底在控制什麼？

---

## 一、先複習四個核心概念

| 概念 | 定義 | 表現 |
|------|------|------|
| 泛化 | 學到規律，能舉一反三 | 有神韻、有靈氣、沒見過但合理的組合 |
| 記憶 | 鎖定訓練集中的具體子集 | 穩定、細節準、但僵硬、難以跳出 |
| 平均 | 找不到明確方向 | 取最安全折衷，平庸、模糊、沒細節、沒個性 |
| 原型 | 潛在空間中的高概率吸引子 | 用特徵組合取代標籤，落在區域內 |

- 有靈氣的創新性生成落在**泛化區間**
- 高一致性、可復現的角色/風格依賴於**記憶區間**
- 不盡人意、不合理、荒誕的圖像（模糊、平庸）落在**平均區間**

---

## 二、光照的六個層級（L1-L6）

這是根據模型能否處理劃分的技術層級：

| 層級 | 現象 | 模型能做到嗎 | 本質 |
|------|------|--------------|------|
| L1 | 直接光照（主光源方向） | ✅ | 完全可控，語義層級 |
| L2 | 漫射光、環境光 | ✅ | prompt + LoRA 可控，語義 + 少量統計 |
| L3 | 月光在水面的質感 | ✅ | 可泛化，散射 + 氛圍，有穩定數據參考 |
| L4 | 丁達爾效應、SSS、邊緣光 | ⚠️ | 部分可行但易退化，體積光學，需要精確透射與遮擋關係 |
| L5 | 鏡面反射（水面倒影、金屬、玻璃） | ❌ | 部分可做（鏡像不對稱）但幾何不準，需要精確幾何對應 |
| L6 | 遞迴反射（鏡中鏡、多重反射） | ❌ | 幾乎做不到，訓練數據中幾乎沒有 |

---

## 三、為什麼丁達爾效應（L4）比月光水面（L3）更難？

這是一個非常反直覺的現象，但數學上非常清晰。

**月光水面（L3）**：模型只需要學習一個統計規律——「冷色調光源 + 波紋表面 → 特定高光分佈」。這是一個紋理/色調的條件分佈，不要求精確幾何對應。模型可以用它的統計能力輕鬆泛化。

**丁達爾效應（L4）**：光柱的可見性依賴於光源位置、遮擋物形狀、觀察角度、介質密度
**四者的精確幾何關係**。這不是「紋理」，而是光線在 3D 空間中的投射與衰減。模型沒有 3D 物理引擎，只能用 2D 圖像中的「共現模式」去猜。
訓練數據中，丁達爾效應的樣本數量少且角度單一，模型學不到它的變換規律，只能記住幾個常見構圖。一旦 prompt 要求的視角超出這些構圖，模型就會坍塌到平均區域——生成模糊的光霧，而不是精確的光柱。

> **結論**：模型能泛化的上限，不是「現象有多複雜」，而是「現象能否用統計規律近似」。月光水面可以，丁達爾效應幾乎不行。

## 四、為什麼 L5 做得到，L6 做不到？

**鏡面反射（L5）**——訓練數據中，鏡子 + 人的圖片大量存在。但模型只記住了「鏡子裡要有東西」，沒記住「那個東西的姿勢必須和外面一致」。
> 結果：倒影像，但姿勢不對，或乾脆歪掉。

**遞迴反射（L6）**——訓練數據中幾乎不存在「鏡中鏡」的精確幾何對應樣本。模型沒有學到「反射的反射必須滿足光學一致性」這個規則。
> 結果：要嘛生成混亂的視覺疊加，要嘛直接忽略，只畫一面鏡子。

> L3-L4 只需要「像」，L5-L6 需要「對」。模型擅長前者，不擅長後者。

## 五、為什麼平均是我們最不想要的結果？

這是模型行為的核心機制。

### 模型是怎麼生成圖片的？

1. 從噪聲開始
2. 根據 prompt，逐步坍塌到潛在空間的某個區域
3. 如果 prompt 精確，它會坍塌到原型區域（泛化）或記憶區域（穩定）
4. 如果 prompt 模糊或衝突，它找不到明確方向，就會坍塌到平均區域

**平均區域長什麼樣？** 把各種相似特徵混在一起，取一個最安全、最不犯錯的版本。
> 結果：穩定、低風險，但平庸、模糊、沒細節。

### 平均不是泛化和記憶的中間值

很多人以為平均是泛化和記憶的中間值，但實際上是：

| 區域 | 自由度 | 細節清晰度 | 可復現性 | 風險 |
|------|--------|------------|----------|------|
| 泛化（原型） | 高 | 高 | 中 | 不穩定 |
| 記憶 | 低 | 極高 | 極高 | 僵化、過擬合 |
| 平均 | 極低 | 低 | 中 | 平庸 |

> 平均不是泛化和記憶的中間態，而是一個獨立的低質量區域。平均不是平衡，而是失焦。

## 六、模型參數大小 vs 泛化 / 記憶 / 平均

這是一個反直覺的現象：**參數越大，平均區間反而越大**。

### 理論上的預期

參數越多 → 模型容量越大 → 能記住更多細節 → 泛化空間也應該越大。

### 實際上的現象

> SD3.5、Flux 這類大模型，在 prompt 不夠精確時，更容易產出「平均臉」「平均髮型」「平均構圖」。反而在某些模糊 prompt 下，小模型的結果更散、更有驚喜。

### 為什麼會這樣？

大模型記憶容量膨脹，能記住大量具體樣本。

- 當 prompt 不精確時，模型會激活多個記憶子集，條件分佈 \(p(x|c)\) 變成多峰的。
- 但模型必須輸出一張圖。
- 它不能「不做決定」。在沒有明確高點的情況下，它會輸出條件期望 \(\mathbb{E}[x|c]\)，落在不同原型區域的幾何中心——而這個中心往往不被任何高概率區域覆蓋。

**結果就是：模糊、平庸、無特徵的平均圖。**

這不是泛化能力變強，而是記憶能力變強。當模型不知道你要哪個具體子集時，它會給你一個最安全的平均值——因為它見過太多可能性了。

> 因此，大模型對 prompt 精度的要求更高。

## 七、seed 在泛化、平均、記憶中的角色

### seed 控制什麼？

seed 決定初始噪聲在潛空間中的位置。根據潛空間退化理論，潛空間可以分解為：

\[\mathcal{Z} = \bigcup_{s \in \mathcal{S}} \mathcal{F}_s\]

- \(\mathcal{S}\)：語義空間（低維）
- \(\mathcal{F}_s\)：對應語義 \(s\) 的纖維——所有生成語義 \(s\) 的潛變數的集合

- **纖維內部方向**：改變 seed，語義不變（冗餘方向）
- **纖維之間的橫截方向**：改變 prompt 精度，語義改變

### 實用的 seed 策略（基於退化理論）

1. 先固定一個 seed，用精確 prompt 錨定目標語義區域
2. 在固定 prompt 下，生成多組 seed 樣本，觀察：
   - 若輸出高度一致 → 當前落在粗纖維內部（語義穩定，但缺乏變化）
   - 若輸出在少數幾個語義之間跳躍 → 當前落在纖維邊界附近（可尋找中間態）
3. 找到「最靈活」的 seed 後，將其作為該 prompt 的基準種子，後續 prompt 微調時固定它
4. 不建議大量隨機 seed 抽獎：絕大多數隨機擾動落在纖維方向，浪費算力且不增加語義多樣性

> seed 不是用來撞運氣的，是用來在退化幾何中尋找橫截路徑的。

## 八、一張圖看懂整個框架（數學版）
```

                    精確 prompt ──→ 模型鎖定到單一高概率區域
                    模糊 prompt ──→ 模型激活多個高概率區域
                    
┌─────────────────────────────────────────────────────────────┐
│                      潛空間 Z                               │
│                                                             │
│   ┌──────────────┐    ┌──────────────┐    ┌─────────────┐ │
│   │   記憶點      │    │  原型區域    │    │  平均區域   │ │
│   │ (狄拉克點)   │    │  (纖維F_s)   │    │ (期望 E[x]) │ │
│   ├──────────────┤    ├──────────────┤    ├─────────────┤ │
│   │ 復現訓練樣本 │    │ 泛化採樣區   │    │ 多峰中心   │ │
│   │ 僵硬、準確   │    │ 有靈氣、新鮮 │    │ 平庸、模糊  │ │
│   └──────────────┘    └──────────────┘    └─────────────┘ │
│                                                             │
│   語義纖維方向 ←→ 橫截方向 (改變prompt精度)                 │
└─────────────────────────────────────────────────────────────┘

光照層級對應的潛空間位置：
- L1-L3：位於原型區域內部
- L4（丁達爾效應）：位於原型區域邊緣，易滑向平均
- L5-L6：所需幾何精度超出原型區域覆蓋範圍
```

### 數學解釋

模型參數 \(\theta\) 凍結後，潛空間的幾何結構就固定了：

- **泛化** = 採樣落在原型區域 \(\mathcal{F}_s\) 內部
- **記憶** = 採樣落在訓練樣本對應的孤立高概率點附近
- **平均** = 由於 prompt 不精確，條件分佈 \(p(x|c)\) 的期望落在一個不被高概率區域覆蓋的位置
- **原型** = 纖維 \(\mathcal{F}_s\) 中機率密度最高的連通子集

---

## 結語

- 你畫不好，不是模型不會畫，而是你沒讓它知道該落在潛在空間的哪個位置。
- 模型參數越大，越需要你把 prompt 寫得精確——否則它會用平均來懲罰你的模糊。
- 光照的上限，不是模型能力的上限，而是「原型是否存在」的上限。
- seed 不是隨機變數，是用來導航退化幾何的工具。
