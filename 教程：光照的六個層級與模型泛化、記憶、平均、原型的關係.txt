一、先複習四個核心概念

概念 定義 表現
泛化： 學到規律，能舉一反三，有神韻、有靈氣、沒見過但合理的組合
記憶： 鎖定訓練集中的具體子集，穩定、細節準、但僵硬、難以跳出
平均： 找不到明確方向，取最安全折衷，平庸、模糊、沒細節、沒個性
原型： 潛在空間中的高概率吸引子，用特徵組合取代標籤，落在區域內

好圖在泛化區間，穩定圖在記憶區間，不盡人意的圖在平均區間


二、光照的六個層級(L1-L6)

這是根據 模型能否處理劃分的技術層級：

層級 現象 模型能做到嗎 本質
L1 直接光照(主光源方向)，完全可控，語義層級
L2 漫射光、環境光， 可控(prompt + LoRA)，語義 + 少量統計
L3 邊緣光、SSS、丁達爾效應 LoRA 可補，散射類，有原型
L4 月光在水面的質感，可以泛化，散射 + 氛圍，有穩定原型
L5 鏡面反射(水面倒影、金屬、玻璃)部分可以，但幾何不準、需要精確對應，原型不存在
L6 遞迴反射(鏡中鏡、多重反射)，幾乎做不到，訓練數據中幾乎沒有


三、為什麼 L4 做得到，L5 做不到？

月光在水面(L4)——屬於 “泛化” 在訓練數據中有大量 水面 + 月光的組合

模型學到的是：水面光澤的質感 + 月光的色調，這是一種組合規律，不是精確對應。
結果：落在原型區域內，有神韻、有靈氣。

鏡面倒影(L5)——需要記憶或規則鏡中角色的倒影不是質感，而是幾何對應

模型沒有學到物體與其反射必須在空間上對齊這個規則。
訓練數據中，大部分 鏡子 + 人的圖片，人與倒影是對齊的——但模型只記住了“鏡子裡要有東西”，沒記住“那個東西的姿勢要和外面一致”。
結果：落在平均區間——倒影像，但不對，或出錯。
L4 能做好，是因為它有原型。L5 做不好，是因為它沒有原型，模型只能用平均來應付。

 
四、為什麼 平均是最不盡人意的結果？

這是模型行為的核心機制，模型是怎麼生成圖片的？
1. 從噪聲開始
2. 根據 prompt，逐步坍塌到潛在空間的某個區域
3. 如果 prompt 精確，它會坍塌到原型區域(泛化)或記憶區域(穩定)
4. 如果 prompt 模糊或衝突，它找不到明確方向，就會坍塌到平均區域

平均區域長什麼樣？

把各種相似特徵混在一起，取一個最安全、最不犯錯的版本。但結果：穩定、低風險、但平庸、模糊，沒細節。

為什麼平均不是 中間值？很多人以為平均是泛化和記憶的中間值，但實際是：

泛化：落在原型區域內，有自由度 → 有靈氣
記憶：落在精確子集內，沒自由度 → 僵硬
平均：落在 什麼都不是的區域 → 平庸

平均不是平衡，而是失焦。
 

五、模型參數大小 vs. 泛化/記憶/平均

這是一個反直覺的現象：參數越大，平均區間反而越大。
理論上的預期，參數越多 → 模型容量越大 → 能記住更多細節 → 泛化空間也應該更大。
實際上的現象模型(SD3.5、Flux)在 prompt 不夠精確時，更容易產出、平均臉、平均髮型，平均構圖 反而有時候在模糊 prompt 下，結果更散，但也更有驚喜。

為什麼會這樣？模型的大小對訓練數據的處理
結果：
以前的模型 容量有限，只能學到大致規律泛化為主，但不穩定，容易散 大模型 容量大，能記住大量具體子集當 prompt 不精確時，默認取這些子集的平均。
模型不是泛化能力變強，而是記憶能力變強。當它不知道你要哪個具體子集時，它會給你一個最安全的平均值——因為它見過太多可能性了。


六、seed 在泛化、平均、記憶中的角色

seed 是很多人忽略但非常重要的參數。seed 控制什麼？

seed 決定初始噪聲的狀態 同樣的 prompt + 同樣的 seed = 同樣的結果(理論上)

seed與三種區間的關係 場景 seed 的作用 說明
在原型區域內(泛化) seed 影響， 神韻的具體表現 同樣的特徵組合，不同 seed 會產生不同但都對味的結果
在記憶區域內 seed 影響不大，結果高度穩定 模型被鎖死在特定子集，seed 只影響微小細節
在平均區域內 seed 影響很大，但結果都不同 seed 會產生不同方向的平均，但都不突出

一個實用的 seed 策略(理論上)

1. 先用精確特徵鎖定原型區域
2. 固定 seed，調整 prompt 直到滿意
3. 最後才微調 seed，尋找 最有神韻的那個版本

seed不是用來撞運氣的，是用來在原型區域內做微調的。

你畫不好，不是模型不會畫，而是你沒讓它知道該落在潛在空間的哪個位置。
模型參數越大，越需要你把 prompt 寫得精確——否則它會用平均來懲罰你的模糊。
光照的上限，不是模型能力的上限，而是 原型是否存在的上限。
